# Guestbook Kubernetes Deployment 
[![Docker](https://img.shields.io/badge/Docker-Ready-blue?logo=docker)](https://www.docker.com/)
[![Kubernetes](https://img.shields.io/badge/Kubernetes-Hands--On-blue?logo=kubernetes)](https://kubernetes.io/)

## Overview

This project demonstrates deploying a **Go-based Guestbook app** to Kubernetes using:
- Docker for containerization
- Kubernetes Deployment for pods
- Horizontal Pod Autoscaler (HPA) for scaling under load
- Load simulation using BusyBox
